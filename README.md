در این پروژه از کتابخانه Java Swing برای طراحی رابط گرافیکی استفاده شده است.
در کلاس Main، اجرای برنامه با ساخت یک شیء از کلاس BMIFrame آغاز می‌شود.
کلاس BMIFrame از JFrame ارث‌بری کرده و به‌عنوان فریم اصلی برنامه عمل می‌کند. در این فریم دو پنل مجزا (IntroPage و BMIPage) تعریف شده که هر دو هم‌زمان به فریم اضافه شده‌اند و با استفاده از متدهای showIntro و showBMI نمایش آن‌ها مدیریت می‌شود.
کلاس IntroPage یک پنل معرفی ساده است که شامل عنوان برنامه و دکمه شروع می‌باشد و با کلیک روی دکمه، کاربر به صفحه محاسبه هدایت می‌شود.
کلاس BMIPage پنل محاسبه BMI است که ورودی قد و وزن را از کاربر دریافت کرده و با استفاده از رویداد دکمه، محاسبه انجام می‌شود. نتیجه محاسبه در همان صفحه نمایش داده می‌شود و امکان بازگشت به صفحه معرفی نیز وجود دارد.
منطق محاسبه BMI در کلاس جداگانه BMILogic پیاده‌سازی شده است تا محاسبات از رابط گرافیکی جدا باشد. در این کلاس، قد از سانتی‌متر به متر تبدیل شده و سپس BMI و وضعیت بدنی محاسبه می‌شود.<img width="1920" height="1080" alt="Screenshot 2025-12-19 221304" src="https://github.com/user-attachments/assets/1e34f1fa-5a05-4dfc-b326-bf08c39e91d6" />
